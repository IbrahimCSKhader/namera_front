# Namira Frontend

React + TypeScript + Vite frontend foundation for Namira.

## Run

```powershell
npm install
npm run dev
```

## Environment

Set the backend base URL with:

```powershell
$env:VITE_API_BASE_URL="http://localhost:5074/api"
```

If the variable is not set, the app uses `http://localhost:5074/api`.
